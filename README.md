# Google 小恐龍遊戲道具改編版
基於 Pygame 的 Google 小恐龍遊戲，**加入自己設計的特殊道具**。

#### 需安裝 Pygame
```pip install pygame```

### 不同版本
- dinosaur-multi.py\
  使用多塊碰況盒近似小恐龍與障礙物的版本
- dinosaur-mask.py\
  繼承使用Pygame.Sprite中的圖像二維矩陣遮罩方法，得到更精確的碰撞判斷結果。
- dinosaur.py\
  最終完整版遊戲
